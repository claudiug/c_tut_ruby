ZenTest
ZenTest contains autotest, which we'll be using to automatically run the tests while we're developing

hoe
Hoe abstracts gem specifications for us. It knows how to properly build a gemspec, and provides us with a few rake tasks that make development simple.

rake-compiler
This gem provides us with compilation tasks, and generally makes building native gems easier. We'll be looking further in to rake-compiler's capabilities in later articles.
